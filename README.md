Creating a dataset


Avant de lancer le script !!
1. mettre le bon numéro de label pour la posture courante
2. mettre le START_INDEX au n° du dernier image_index+1 enregistré dans 'hand_data.csv'
3. porter un vêtement jusqu'au poignet pour faciliter la segmentation avec SAM


Après avoir lancer le script !!
1. être sûr de faire la bonne pose
2. avoir tout les landmarks de la main présent sur l'image
3. appuyer sur S pour enregistrer les coord des lm et l'image 
4. prendre des photos avec différentes rotations, en bougeant la main dans toute l'image et en faisant des zoom et des dézoom
5. appuyer sur Q pour quitter
